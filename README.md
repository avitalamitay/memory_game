#**Welcome to my memory game!**

##**Special features**
- **Theme selection**: Choose between 3 custom themes, or select "Random" for a randomised choice. Custom layout per theme.
  - **Harry Potter**: Fetches character images and names.
  - **Dogs**: Fetches cute dog pictures.
  - **Flags**: Fetches worldwide country flags and displays their names below, keeping the flag image contained neatly at the top and showing the country's official name right below it
- **In-Game Sidebar**: Displays the currently active theme and includes a "Start Over" safety button to return to the main menu at any point.
- **Victory Screen**: Displays a celebration screen ("You Won!") upon matching all pairs, with a fully responsive "Play Again" loop.
  
##**Technologies used**
- html
- css
- JavaScript

##**APIs Integrated**
- **Dogs API** - https://dog.ceo/dog-api/documentation/random → Dogs pictures free public api
2. **Harry Potter API** - https://hp-api.onrender.com/ → Harry Potter free public api
3. **Countries and flags API** - https://restcountries.com/ → Countries flags free public api

##**How to run?**
- Download the files (make sure all files are in the same directory)
- Open the html file
- Select your preferred theme and start playing!
